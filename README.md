# AndroidUtils
Android Common Utils

##使用
在项目`build.gradle`中添加


```

    dependencies {
        compile fileTree(include: ['*.jar'], dir: 'libs')
        testCompile 'junit:junit:4.12'
        compile 'com.android.support:appcompat-v7:23.1.1'
        ...
        compile 'com.cjs:utils:0.0.2'
        ...
    }

```
